# Personal Portfolio – Deepak Singh Kanyal

A clean and responsive personal portfolio website built using HTML and CSS. This website showcases my profile, skills, projects, and contact information in a simple and aesthetically pleasing layout.

---

## - Features

- Fully responsive layout
- Modern and clean UI design
- Dedicated sections: Home, About, Projects, Resume & Contact
- Smooth scrolling navigation
- Structured project cards
- Simple & fast—no frameworks used

---

## - Sections Overview

### 1. Home
```
A hero section introducing me as Deepak Singh Kanyal, a frontend learner and web design enthusiast.
```

### 2. About
```
A brief introduction about my background, interests, and goals as a developer.
```

### 3. Projects
```
A grid-styled section showcasing project cards.
```

### 4. Resume
```
Clickable button to open or download my resume.
```

### 5. Contact
```
Provides multiple ways to connect: email, social links, etc.
```
